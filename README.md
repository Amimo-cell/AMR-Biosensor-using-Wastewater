# AMR-Biosensor-using-Wastewater
Sentinel-Sewer: AI-powered simulation of CRISPR-based biosensors for real-time antimicrobial resistance detection in wastewater networks. Early warning system for public health outbreaks.
A computational simulation framework for CRISPR-based electrochemical biosensors** that detect antimicrobial resistance (AMR) genes in municipal wastewater. This system models city-wide IoT sensor networks to provide early warnings of AMR outbreaks 7-14 days before clinical cases appear.
